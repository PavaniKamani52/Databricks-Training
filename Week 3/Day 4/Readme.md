# WEEK 3 – DAY 4
## RANK() Functions in MySQL

This project demonstrates the implementation of the **RANK()** window function in MySQL using employee and order datasets. The examples focus on ranking records for business analysis and reporting purposes.

### Queries Demonstrated

- City-wise order ranking based on order amount
- Department-wise salary ranking
- Employee name ranking in alphabetical order

## Topics Covered

### 1. City-wise Order Ranking
Assign ranks to orders within each city according to the order amount.

### 2. Department-wise Salary Ranking
Rank employees within each department based on salary in ascending order.

### 3. Employee Name Ranking
Generate rankings for employees alphabetically by their names.

## SQL Concepts Used

- `RANK()`
- `OVER()`
- `PARTITION BY`
- `ORDER BY`
- Window Functions

By completing this project, you will learn:
