# WEEK 3 – DAY 6
## DENSE_RANK() Functions in MySQL

This project demonstrates the implementation of the **DENSE_RANK()** window function in MySQL using employee and order datasets. The examples focus on ranking data without gaps, making it useful for analytical and business reporting scenarios.

## Queries Demonstrated

- Employee joining date ranking
- Order amount ranking
- City-wise dense ranking based on order amount

## Topics Covered

### 1. Dense Ranking Based on Joining Date
Assign dense ranks to employees according to their joining dates.

### 2. Dense Ranking Based on Order Amount
Rank orders based on the highest order amount using dense ranking.

### 3. City-wise Dense Ranking
Generate dense ranks within each city according to order amounts.
