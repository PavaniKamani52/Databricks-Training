# Week 2 - Day 7: NULL Functions & String Functions in MySQL

## Project Overview

This project focuses on mastering NULL Handling and String Manipulation functions in MySQL through practical business scenarios. It demonstrates how SQL can be used to manage missing values, clean and transform text data, perform financial calculations, and prepare datasets for reporting and analysis.

The project uses employee, product, and order management examples to simulate real-world database operations commonly encountered in business applications.

## Key Features

### NULL Functions

* Detecting missing values using `IS NULL` and `IS NOT NULL`
* Replacing NULL values with default values using `IFNULL()`
* Implementing fallback logic with `COALESCE()`
* Preventing calculation errors using `NULLIF()`
* Counting and validating records with `COUNT()`
* Handling incomplete business data efficiently

### String Functions

* Text length calculations
* Case conversion and normalization
* Removing unwanted spaces
* String concatenation and formatting
* Substring extraction
* String searching and replacement
* Text reversal and padding
* Email and text parsing operations
